TWEETSNEK - Python twitter scraping

1. Create/Choose Twitter account for scraping
2. Enter key/token in tweetsnek.py
3. Choose User/User ID of person that can send DM's to control scraping in tweetsnek.py
4. Send commands to control which tweets are saved

## CONTROL TWEET FORMAT
PARSESIG {replace/add/remove/stop} {optional: KW1::KW2}
